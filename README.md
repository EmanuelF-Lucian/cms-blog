# Laravel 12 Inertia Vue Blog

A simple blog application built with the Laravel 12 starter kit using Inertia.js + Vue.  
Authentication is included out of the box.

The project is structured around two roles:

- **Admin** – manages the platform through Filament
- **User** – can read and manage their own posts after authentication

---

## Tech Stack

- Laravel 12
- Inertia.js
- Vue 3
- TailwindCSS
- Filament (admin panel)
- Laravel Starter Kit auth scaffolding

---

## Core Features

### Authentication

- Login / Register / Logout
- Protected routes
- Only authenticated users can access the content

### Admin Panel (Filament)

- Full post management
- User management
- Platform administration tools

### User Dashboard

- Profile management
- Create posts
- Edit own posts
- Update own posts
- Delete own posts
- Cannot modify other users' content

### Blog

- Authenticated users can:
    - View all posts
    - Open single post pages
    - Like posts
    - Comment on posts
